Bootstrap Tables 
================
A table is used to represent our data in the form of rows and columns.

A "table" class is used to represent bootstrap table.

ex:1
-----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>

		<!-- add bootstrap plugins -->
		<meta charset="utf-8">
    	
    	<meta name="viewport" content="width=device-width, initial-scale=1">

    	<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" >	

 		<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"> </script>

	</head>
	<body>
			<div class="container mt-5">
				
				<table class="table">
					
					<tr>
						<th>ID</th>
						<th>NAME</th>
						<th>EMAIL</th>
					</tr>
					<tr>
						<td>101</td>
						<td>Alan</td>
						<td>alan@gmail.com</td>
					</tr>
					<tr>
						<td>102</td>
						<td>Jose</td>
						<td>jose@zoho.com</td>
					</tr>
					<tr>
						<td>103</td>
						<td>Nelson</td>
						<td>nelson@outlook.com</td>
					</tr>
					<tr>
						<td>104</td>
						<td>Jack</td>
						<td>jack@yahoo.com</td>
					</tr>

				</table>	

			</div>
	</body>
</html>

Bootstrap List 
=================
Bootstrap list is used to represent list of items.

A "list-group" class is used to represent unorder list.

A "list-group-item" class is used to represent list of item.

ex:1
-----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>

		<!-- add bootstrap plugins -->
		<meta charset="utf-8">
    	
    	<meta name="viewport" content="width=device-width, initial-scale=1">

    	<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" >	

 		<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"> </script>

	</head>
	<body>
			<div class="container mt-5">
				
				<ul class="list-group">
					<li class="list-group-item">ReactJS</li>
					<li class="list-group-item">AngularJS</li>
					<li class="list-group-item">VueJS</li>
					<li class="list-group-item">Backbone</li>
				</ul>	

			</div>
	</body>
</html>

Bootstrap alerts
==============
A "alert" class is used to represent bootstrap alert.

We can use alert class along with contextual classes.

ex:
	alert-danger
	alert-info
	alert-success
	alert-warning
	and etc.

ex:1
----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>

		<!-- add bootstrap plugins -->
		<meta charset="utf-8">
    	
    	<meta name="viewport" content="width=device-width, initial-scale=1">

    	<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" >	

 		<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"> </script>

	</head>
	<body>
			<div class="container mt-5">
				
				<div class="alert alert-danger">
					<strong>Danger!</strong> Please don't try this.
				</div>
				<div class="alert alert-info">
					<strong>Alert!</strong> Think twice before you take any decision.
				</div>
				<div class="alert alert-warning">
					<strong>Warning!</strong> Keep a distance.
				</div>
				<div class="alert alert-success">
					<strong>Yahoo!</strong> You transaction done successfully.
				</div>


			</div>
	</body>
</html>

Bootstrap Buttons 
==================
The .btn class is intended to be used in conjunction with our button variants, 
or to serve as a basis for your own custom styles.

WE need to use btn along with contextual classes
ex:
	btn-primary
	btn-info
	btn-danger
	and etc.

A .btn-outline-* ones to remove all background images and colors on any button.

ex:1
----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>

		<!-- add bootstrap plugins -->
		<meta charset="utf-8">
    	
    	<meta name="viewport" content="width=device-width, initial-scale=1">

    	<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" >	

 		<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"> </script>

	</head>
	<body>
			<div class="container mt-5">
				
				
				<button class="btn btn-primary">primary</button>
				<button class="btn btn-secondary">secondary</button>
				<button class="btn btn-info">info</button>
				<button class="btn btn-warning">warning</button>
				<button class="btn btn-danger">danger</button>
				<button class="btn btn-success">success</button>
				<button class="btn btn-dark">dark</button>

			</div>
	</body>
</html>
